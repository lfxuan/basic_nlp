# LaTex 规则

1. 行内公式 (Inline Math)
   语法：$公式内容$
   示例：$a^2 + b^2 = c^2$

   

2. 公式块 (Math Blocks)
   如果你需要编写独立成行、复杂的公式，可以使用两对美元符号 $$ 并回车

3. 常用 LaTeX 语法示例

4. | **效果**  | **LaTeX 代码**                               |
   | --------- | -------------------------------------------- |
   | 分数      | `\frac{分子}{分母}`                          |
   | 上标/下标 | `x^2`, `a_i`                                 |
   | 根号      | `\sqrt{x}` 或 `\sqrt[n]{x}`                  |
   | 累加/累乘 | `\sum_{i=1}^n`, `\prod`                      |
   | 希腊字母  | `\alpha`, `\beta`, `\gamma`, `\pi`           |
   | 矩阵      | `\begin{matrix} a & b \\ c & d \end{matrix}` |

   

5. 相关集合运算符

   | **符号**        | **LaTeX 代码**               | **含义**           |
   | --------------- | ---------------------------- | ------------------ |
   | $\in$           | `\in`                        | 属于               |
   | $\notin$        | `\notin`                     | 不属于             |
   | $\ni$           | `\ni`                        | 包含于（反向属于） |
   | $\subseteq$     | `\subseteq`                  | 包含于（子集）     |
   | $\varsubsetneq$ | `\varsubsetneq`              | 真子集             |
   | $\emptyset$     | `\emptyset` 或 `\varnothing` | 空集               |

   

6. 常见的计算运算符

   | **符号名称**  | **效果**  | **LaTeX 代码**  | **示例**                  |
   | ------------- | --------- | --------------- | ------------------------- |
   | **叉乘**      | $\times$  | `\times`        | `$3 \times 5 = 15$`       |
   | **点乘**      | $\cdot$   | `\cdot`         | `$\vec{a} \cdot \vec{b}$` |
   | **卷积/星号** | $\ast$    | `\ast`          | `$f(t) \ast g(t)$`        |
   | **除号**      | $\div$    | `\div`          | `$6 \div 3 = 2$`          |
   | **正负号**    | $\pm$     | `\pm`           | `$x = \pm 1$`             |
   | **负正号**    | $\mp$     | `\mp`           | `$a \mp b$`               |
   | **不等于**    | $\neq$    | `\neq` 或 `\ne` | `$a \neq b$`              |
   | **约等于**    | $\approx$ | `\approx`       | `$\pi \approx 3.14$`      |
   | **恒等于**    | $\equiv$  | `\equiv`        | `$a \equiv b \pmod{n}$`   |

   

7. 相关符号对照

   | **符号**                | **LaTeX 代码**          | **含义**                   |
   | ----------------------- | ----------------------- | -------------------------- |
   | $\ll$                   | `\ll`                   | 远小于 (Much less than)    |
   | $\gg$                   | `\gg`                   | 远大于 (Much greater than) |
   | $\langle \dots \rangle$ | `\langle \dots \rangle$ | 单角括号 (常用作内积)      |
   | $\subset\subset$        | `\subset\subset`        | 紧包含                     |

   

8. 常用空格命令（按间距从小到大排序）

   | **效果**    | **LaTeX 代码** | **说明**                                    |
   | ----------- | -------------- | ------------------------------------------- |
   | $a\,b$      | `\,`           | **最常用**：微小间距（通常用于 $dx$ 前）    |
   | $a\:b$      | `\:`           | 中等间距                                    |
   | $a\;b$      | `\;`           | 较大间距                                    |
   | $a\ b$      | `\ `           | 反斜杠+空格：普通单词间距                   |
   | $a\quad b$  | `\quad`        | **最推荐**：相当于当前字体尺寸的空格（1em） |
   | $a\qquad b$ | `\qquad`       | 两个 `\quad` 的间距（2em）                  |
   | $a\!b$      | `\!`           | **负间距**：让字符靠得更近（用于微调重叠）  |
